# Forage CommonWealth Bank Introduction to Data Science Job Simulation
Commonwealth Bank Introduction to Data Science Job Simulation on Forage - January 2026
  1. Completed a job simulation involving Data Management skills for Commonwealth Bank's Data Science team.
  2. Demonstrated proficiency in creating data engineering pipelines to aggregate and extract valuable insights from datasets, optimizing data-driven decision-making.
  3. Acquired skills in anonymizing personal data within datasets, ensuring compliance with data privacy regulations.
  4. Proposed effective data analysis approaches, particularly related to social media, and demonstrated the ability to design well-structured databases for efficient information management.

# Task One: Data Aggregation and Analysis
Create a data engineering pipeline to aggregate and extract insights from a data set
What you'll learn
  1. Understand the importance of data engineering and analysis in a long-term data-focused vision.
  2. Gain experience in using real-world transactional data for analysis.
  3. Learn to perform data analysis and answer specific questions based on the dataset provided.
     
# Task Two: Data Anonymisation
Anonymise personal data within a data set
What you'll learn
  1. Understand the importance of data anonymization in protecting sensitive customer information.
  2. Learn various techniques for anonymizing data while preserving its utility.
  3. Comprehend the risks associated with linkage attacks on anonymized data.

# Task Three: Propose Data Analysis Approaches
Explore data insights that can be unlocked with social media
What you'll learn
  1. Understand the difference between structured and unstructured data, with a focus on unstructured data from social media.
  2. Explore the types of data that can be extracted using the Twitter API and the potential insights that can be derived from them.
  3. Gain knowledge about how to use Twitter data for business purposes and decision-making.

# Task Four: Designing a Database
Design the structure of information in a database
What you'll learn
  1. Understand the principles of database design, including reducing redundancy and ensuring data integrity.
  2. Familiarize with Microsoft's guidelines for effective database design.
